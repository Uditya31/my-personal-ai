# My Personal AI - Full Stack Application

A production-ready full-stack AI Assistant application built with React, FastAPI, and PostgreSQL.

## 🚀 Features

- **Authentication**: JWT-based authentication with refresh tokens
- **AI Chat Interface**: ChatGPT-style interface with streaming responses
- **Conversation Management**: Create, rename, delete, and search conversations
- **File Upload**: Support for PDF and DOCX files with text extraction
- **Markdown & Code Highlighting**: Beautiful rendering of AI responses
- **Dark/Light Mode**: Theme switching support
- **Responsive Design**: Mobile-first UI design
- **Advanced Features**: Voice input, text-to-speech, image generation
- **Rate Limiting**: Protected against abuse
- **Security**: Input validation, environment variables, API key protection

## 📋 Tech Stack

### Frontend
- ReactJS (Vite)
- Tailwind CSS
- Axios
- React Router

### Backend
- Python 3.9+
- FastAPI
- SQLAlchemy
- PostgreSQL / SQLite
- OpenAI API
